A polished concept dashboard demonstrating organized, client-facing web project management for a digital agency environment.

## What This Demonstrates

- **Client Communication Tracking** — Organized request intake with status triaging
- **Developer Task Translation** — Converting client feedback into actionable dev tasks
- **Project Health Monitoring** — At-a-glance health indicators across all active projects
- **Risk Visibility** — Developer blockers and at-risk items surfaced immediately
- **Launch Readiness** — Pre-deployment checklists ensuring nothing gets missed

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons
- Static mock data (no backend)
- Deployed on Vercel

## Pages

1. **Login** — Demo credentials with split-screen branding
2. **Dashboard** — KPIs, project health, blockers, milestones, launch readiness
3. **Project Detail** — Full project view with Client → Developer Translation
4. **Client Requests** — Card-based request management with status filters
5. **Launch Readiness** — Pre-launch checklists with progress tracking

## Purpose

Built as a working concept to accompany a Web Development Project Manager application. Demonstrates understanding of agency workflows, client management, and project delivery processes.
